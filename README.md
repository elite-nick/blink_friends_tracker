# BLINK Map Friends Tracker

<img width="1088" height="543" alt="Blink_PC_logo" src="https://github.com/user-attachments/assets/bb7d9abd-f533-463d-b1c1-d450689b95a5" />

Python-скрипт для показа местоположения и прочей информации **всех ваших друзей** прямо на ПК (где можно запустить Python и есть интернет конечно)! Работает по API.

## Возможности скрипта

- Показ местоположения ваших друзей
- Вместе с логином друзей показывается их ID, точность местоположения, тип места, заряд батареи, тип их текущего места, текущий хвост маркера, **дата последнего онлайна**, дата с момента нахождения в месте друга и последнего обновления координат
- Увидеть полный список друзей вашего друга
- Отправить **любой** аудио-стикер с **любым** количеством (нужно знать ID стикера)
- Увидеть полный JSON с информацией вашего друга
- Можно использовать вместе с мобильным приложением
- Ваш последний онлайн не обновляется
- Возможность посмотреть количество шагов за день

## Подготовка к настройке и запуску

- Установите зависимость flask (**pip install requests flask**)
- Скачайте весь этот проект
- В начале файла **main.py** укажите ваш **PROVIDER_IDENTIFIER** и **STATIC_TOKEN** (где достать не подскажу, додумайтесь сами)

## Запуск
После выполнения подготовки просто запустите скрипт (**python main.py**), в любом браузере откройте **http://127.0.0.1:5000/** и следите за друзьями из ПК!

## Решение проблем и FAQ
- Это ПК версия Blink?
- - В какой-то мере да. Можно смотреть где находятся ваши друзья не покидая компьютер. Но тем не менее авторизация все равно нужна через смартфон и нет множества фич с оригинального приложения.
- Вчера скрипт работал нормально, а сегодня уже **Query Error**, что делать?
- - Обновите данные еще раз, обновите вручную сессию или проверьте свои личные токены.
- А можешь добавить **имя_твоей_фичи**?
- - Да, если предложить её в **Issues**. НО, сразу говорю, что фичи по типу бесплатных блинкоинов, бесплатного получения платных хвостов, отправка бесплатно подарков, узнать местоположение не друга **невозможны** и добавлять их не буду!
